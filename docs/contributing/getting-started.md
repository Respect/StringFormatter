# Getting Started

### Prerequisites

- PHP 8.5 or higher
- Composer
- Git

### Setting Up the Development Environment

1. **Fork the repository**

```bash
# Fork the repository on GitHub, then clone your fork
git clone https://github.com/yourusername/StringFormatter.git
cd StringFormatter
```

2. **Install dependencies**

```bash
composer install
```

3. **Run tests**

```bash
composer test
```
